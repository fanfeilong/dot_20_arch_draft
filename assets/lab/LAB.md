# d2a Lab

## Purpose

This directory is a local architecture lab for studying one target project.

Use it to:

1. Decompose the target project's architecture under `docs/architecture/`.
2. Derive a runnable mini implementation under `src/`.
3. Add integration tests under `tests/`.
4. Build a local report experience under `report/`.

## Suggested Workflow

1. Define the target repository and analysis scope.
2. Use the built-in `d2a-*` skills to fill the architecture docs.
3. Extract the one core architecture idea worth reimplementing.
4. Build the mini version in the target stack under `src/`.
5. Add tests that prove the mini version works.
6. Turn the result into a local report.
